## 1. AI / LLM

### Claude Design 출시 (4/17) — AI가 프로토타입/슬라이드를 만든다
Anthropic이 **Claude Design**을 리서치 프리뷰로 출시했다. Opus 4.7 기반으로, **프로토타입, 슬라이드, 앱/웹 인터페이스, 소셜 미디어 에셋** 등 시각적 작업물을 대화형으로 생성하는 도구다.

**핵심 기능:**
- 대화로 요구사항 설명 → 초안 생성 → 대화/주석/직접 편집/슬라이더로 반복 수정
- **회사 코드베이스와 기존 디자인 파일을 읽어** 색상/타이포/컴포넌트를 자동 반영
- PDF, URL, PPTX, **Canva로 내보내기** 가능 (Canva에서 편집 가능)
- Claude Code와의 핸드오프 지원 — 디자인 → 코드로 전환
- **Canva 엔진** 기반으로 동작

**왜 주목할만한가:** Figma를 대체하겠다는 직접적인 도전. 디자이너가 아닌 개발자도 "이런 느낌의 UI 만들어줘"라고 말하면 바로 프로토타입이 나온다. 특히 코드베이스를 읽어서 기존 디자인 시스템에 맞추는 기능이 인상적. 바이브코딩에서 "말로 디자인까지"가 현실화된다.
- [TechCrunch](https://techcrunch.com/2026/04/17/anthropic-launches-claude-design-a-new-product-for-creating-quick-visuals/) | [VentureBeat](https://venturebeat.com/technology/anthropic-just-launched-claude-design-an-ai-tool-that-turns-prompts-into-prototypes-and-challenges-figma)

### NSA가 Claude Mythos 사용 중 — 블랙리스트와 모순
**NSA가 Claude Mythos Preview를 배포하고 있다**는 보도가 나왔다. 아이러니한 것은 NSA를 관할하는 **국방부(Pentagon)가 3월에 Anthropic을 공급망 리스크로 블랙리스트에 올렸다**는 점. 같은 정부 내에서 한쪽은 블랙리스트, 한쪽은 적극 도입하고 있는 모순적 상황.

Anthropic CEO Dario Amodei가 4/17 백악관에서 수석보좌관과 재무장관을 만났으며, "생산적인 논의"였다고 양측이 밝혔다.

**왜 주목할만한가:** Anthropic vs 미국 정부 갈등이 복잡하게 전개되고 있다. 블랙리스트에 올랐지만 NSA는 Mythos를 쓰고, 백악관과는 협상 중. Claude 사용자 입장에서는 이 분쟁이 어떻게 해결되느냐에 따라 서비스 안정성에 영향이 올 수 있으므로 계속 주시할 필요.
- [Decrypt](https://decrypt.co/364894/nsa-anthropic-mythos-ai-white-house-pentagon-supply-chain-risk)

### Snap 감원 1,000명 후폭풍 — "AI가 코드 65%" 충격파 지속
지난주 Snap 감원 소식이 주말 동안 업계에 큰 파장을 일으켰다. 주가 8% 상승, 연간 $5억 비용 절감. **"AI가 코드 65%를 작성"**이라는 수치가 다른 기업들에게도 유사한 구조조정의 명분이 될 수 있다는 우려가 확산.

**왜 주목할만한가:** 이 사건이 업계 트렌드로 확산될 가능성. 다른 기업들도 "우리도 AI가 코드 X%를 작성하니 인력 효율화가 가능하다"는 논리를 사용할 수 있다. 개발자에게 AI 협업 능력이 더 중요해지는 추세가 가속화.

## 2. 보안 — MCP 아키텍처 취약점 (매우 중요)

### MCP 프로토콜 자체에 심각한 취약점 — "1.5억 다운로드 영향"
OX Security가 4/15에 공개한 MCP(Model Context Protocol) 아키텍처 취약점이 주말 동안 큰 화제가 됐다.

**핵심:**
- 코딩 실수가 아닌 **MCP 프로토콜 자체의 구조적 결함**
- **1.5억 다운로드**, **20만+ 공개 서버** 영향
- **10개 이상의 CVE** 발급
- 연구팀이 **실제 운영 중인 6개 플랫폼에서 명령 실행 성공**, 11개 MCP 마켓플레이스 중 **9개에서 보안 검사 우회**
- **영향받는 도구:** Cursor, VS Code, Windsurf, Claude Code, Gemini-CLI 모두 취약
- Windsurf(CVE-2026-30615)는 **사용자 상호작용 없이 악용 가능**

**Anthropic의 입장:** "이 동작은 설계 의도대로이며, 보안 검증은 **개발자 책임**"이라며 프로토콜 수정 거부.

**Palo Alto Unit 42도 별도 보고서 발표:** MCP Sampling을 통한 새로운 프롬프트 인젝션 공격 벡터를 추가로 공개.

**왜 주목할만한가:** 우리가 사용하고 있는 **Claude Code의 MCP 서버도 영향 범위에 포함**된다. context7, serena 등 MCP 서버를 설치해서 쓰고 있다면 주의가 필요하다. Anthropic이 "개발자 책임"이라고 한 만큼, **MCP 서버 선택과 설정에 더 신중해야** 한다. 신뢰할 수 없는 MCP 서버는 설치하지 말고, 기존에 설치된 것도 검토해볼 것.
- [OX Security](https://www.ox.security/blog/the-mother-of-all-ai-supply-chains-critical-systemic-vulnerability-at-the-core-of-the-mcp/) | [Infosecurity Magazine](https://www.infosecurity-magazine.com/news/systemic-flaw-mcp-expose-150/) | [Palo Alto Unit 42](https://unit42.paloaltonetworks.com/model-context-protocol-attack-vectors/)

## 3. 개발 도구 & 트렌드

### GitHub Copilot 학습 데이터 수집 — D-3!
4월 24일까지 **3일 남음**. Copilot Free/Pro/Pro+ 코드가 AI 학습에 사용된다. **Settings > Privacy에서 opt-out. 이번 주 안에 반드시.**

### Apple App Store 새 SDK 요구사항 — D-7
4월 28일부터 iOS 26, iPadOS 26, tvOS 26, visionOS 26, watchOS 26 SDK 필수. 앱 개발팀이라면 빌드 환경 확인.

### .NET Aspire 13.2 — Detached Mode
코딩 에이전트가 AppHost를 **백그라운드에서 시작/검사/관리**할 수 있는 Detached Mode 추가. AI 에이전트와 .NET 개발 워크플로우의 통합이 더 자연스러워졌다.

### General Compute — ASIC 기반 에이전트 추론 클라우드
GPU 대신 **전용 ASIC 칩**으로 AI 에이전트 워크로드를 처리하는 클라우드 플랫폼 발표. 5월 15일 GA 예정.

**왜 주목할만한가:** GPU 품귀가 계속되면서 ASIC 기반 대안이 등장. 에이전트 워크로드에 특화된 인프라가 별도 시장으로 형성되고 있다.
