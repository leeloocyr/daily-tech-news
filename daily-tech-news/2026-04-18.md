## 1. AI / LLM

### Claude Opus 4.7 후속 분석 — 시스템 프롬프트 변경점 공개
어제 출시된 Opus 4.7의 시스템 프롬프트 변경사항을 **Simon Willison**이 Git 히스토리로 비교 분석해 공개했다. HN에서 큰 화제.

**주요 변경:**
- **"먼저 물어보지 말고 일단 시도해"** — 새로운 `<acting_vs_clarifying>` 섹션 추가. "세부사항이 불명확하면 먼저 질문하지 말고 합리적으로 시도하라"는 지침. 가장 논란이 큰 변경
- **새 도구 추가** — Claude in Chrome(브라우징 에이전트), Claude in Excel(스프레드시트 에이전트), Claude in PowerPoint(슬라이드 에이전트) 언급
- **트럼프 관련 문구 제거** — 4.6에는 "트럼프가 현 대통령"이라는 명시적 문구가 있었으나, 4.7은 지식 컷오프가 2026년 1월로 업데이트되면서 불필요해짐
- **Claude Platform으로 명칭 변경** — "developer platform" → "Claude Platform"

**왜 주목할만한가:** "먼저 물어보지 말고 시도해" 변경은 양면이 있다. 바이브코딩할 때 "이거 할까요?" 같은 불필요한 질문이 줄어 속도가 빨라지지만, 중요한 결정을 사용자 확인 없이 진행할 수 있다는 위험도 있다. **CLAUDE.md에 "코드 변경 전 반드시 확인 받을 것"을 명시하는 것이 더 중요해졌다** — 우리 ROI_PROJECT_V2에는 이미 이 규칙이 있으니 OK.
- [Simon Willison](https://simonwillison.net/2026/Apr/18/opus-system-prompt/) | [HN Discussion](https://news.ycombinator.com/item?id=47823270)

### Snap 1,000명 감원 — "AI가 코드의 65% 작성"
Snapchat 모회사 Snap이 **직원 16%(약 1,000명)를 해고**하고 300개 이상의 채용 공고를 폐쇄했다. CEO Evan Spiegel은 "AI가 이제 **새 코드의 65%를 생성**하고 있어 소규모 팀으로도 같은 성과를 낼 수 있다"고 밝혔다. 연간 **$5억 이상** 비용 절감 예상. 발표 당일 주가는 **8% 상승**.

**왜 주목할만한가:** AI로 인한 대규모 감원의 가장 직접적인 사례가 등장했다. "AI가 코드 65%를 작성"이라는 구체적 수치는 업계에 충격을 주고 있다. 이전까지 "AI가 일자리를 대체한다"는 추상적 우려였는데, 이제 실제 수치와 감원 규모가 나왔다. 개발자로서 **AI 도구 활용 능력이 생존 역량**이 되고 있다는 현실적 경고.
- [CNBC](https://www.cnbc.com/2026/04/15/snap-stock-layoffs-16-percent-workforce.html) | [TechCrunch](https://techcrunch.com/2026/04/15/snap-is-cutting-1000-jobs-16-of-its-workforce/)

### Anthropic 매출 $300억 돌파 — OpenAI 추월 가능성
Anthropic의 연간 매출이 **$300억**을 넘어섰다는 보도. 2025년 말 $90억에서 몇 달 만에 3배 이상 폭증. OpenAI를 추월할 가능성이 언급되고 있다. #QuitGPT 이후 Claude로의 대규모 사용자 이동이 주요 원인으로 분석됨.

**왜 주목할만한가:** AI 시장에서 "윤리적 입장 = 수익"이라는 공식이 성립한 사례. 군사 계약을 거부하고 블랙리스트에 올라갔음에도 오히려 사용자가 늘었고 매출이 3배 폭증했다.

### NIST, CVE 처리 방식 변경 — "폭발하는 취약점에 대응 못 함"
미국 NIST가 **국가 취약점 데이터베이스(NVD)의 CVE 처리 방식을 변경**했다. 2020~2025년 CVE 제출이 **263% 증가**하면서 처리 역량이 한계에 달해, 특정 조건을 충족하는 CVE만 선별적으로 분석(enrich)하겠다고 발표.

**왜 주목할만한가:** 보안 취약점이 기하급수적으로 증가하면서 국가 기관조차 처리를 못 따라가는 상황. AI(Claude Mythos 같은)가 자동 취약점 발견을 시작하면 이 수치는 더 폭증할 것. 보안 팀에 AI 기반 자동 분류/우선순위 도구가 필수가 되고 있다.

## 2. 보안

### Apache ActiveMQ 활발한 악용 중
Apache ActiveMQ Classic의 심각한 취약점이 실제 공격에 사용되고 있어 CISA가 경고. 사용 중이면 즉시 패치.

### Microsoft Defender 제로데이 3종 — "BlueHammer, RedSun, UnDefend"
보안 연구자 Chaotic Eclipse가 Microsoft의 취약점 공개 프로세스 불만으로 **Defender 제로데이 3개를 공개**. 로컬 권한 상승(BlueHammer, RedSun)과 서비스 거부 + 정의 업데이트 차단(UnDefend).

**왜 주목할만한가:** MS Defender를 보안 도구로 의존하는 팀이라면 주의. 특히 UnDefend는 Defender의 정의 업데이트를 차단하므로 다른 보안 레이어도 필요.

### Cisco 핵심 취약점 4종 패치
Cisco Identity Services, Webex Services에서 **임의 코드 실행 + 사용자 사칭** 가능한 4개 취약점 패치. Cisco 장비/서비스 사용 중이면 즉시 업데이트.

### Operation PowerOFF — DDoS 서비스 53개 도메인 차단
국제 법집행 기관이 **53개 DDoS 서비스 도메인**을 차단하고 4명을 체포. 75,000명 이상의 사이버 범죄자가 이용했던 상업적 DDoS 서비스.
