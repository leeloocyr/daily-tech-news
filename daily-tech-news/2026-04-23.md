## 1. AI / LLM

### SpaceX, Cursor $600억 인수 옵션 확보 — AI 코딩 전쟁 본격화
SpaceX가 AI 코딩 스타트업 **Cursor를 $600억에 인수할 수 있는 옵션**을 확보했다. 당장 인수하지 않더라도 9개월간의 협업 대가로 **$100억**을 지불한다. Cursor는 즉시 xAI의 슈퍼컴퓨터 **Colossus**(H100 100만 개 규모)에 접근할 수 있게 된다.

**왜 주목할만한가:** Elon Musk가 SpaceX와 xAI를 합병한 후 첫 대형 AI 딜. Cursor는 이미 수십만 개발자가 사용 중인 AI 코딩 도구인데, SpaceX/xAI의 컴퓨팅 파워를 결합하면 OpenAI(Codex)와 Anthropic(Claude Code)에 대한 직접적 경쟁자가 된다. **AI 코딩 도구 시장이 3파전**으로 재편되는 신호.
- [TechCrunch](https://techcrunch.com/2026/04/21/spacex-is-working-with-cursor-and-has-an-option-to-buy-the-startup-for-60-billion/) | [CNBC](https://www.cnbc.com/2026/04/21/spacex-says-it-can-buy-cursor-later-this-year-for-60-billion-or-pay-10-billion-for-our-work-together.html) | [Axios](https://www.axios.com/2026/04/21/spacex-ai-cursor-deal)

### Claude Code, Pro 플랜에서 제거 실험 → 빠른 철회
Anthropic이 **Claude Code를 $20 Pro 플랜에서 제거**하는 실험을 진행했다가 빠르게 철회했다. 신규 가입자 2%를 대상으로 한 A/B 테스트였으나, 커뮤니티 반발이 거셌다. Anthropic 성장 담당자가 "기존 Pro/Max 구독자는 영향 없음"을 해명하고, 가격 페이지와 문서를 원상 복구.

**왜 주목할만한가:** Claude Code의 인기가 너무 높아 Anthropic의 인프라 비용 부담이 커지고 있다는 신호. GitHub도 Copilot 가입을 일시 중단할 만큼 AI 코딩 도구의 수요가 공급을 초과하는 상황. **Max 플랜($100/월)으로의 업셀 압박이 올 수 있으니 주시 필요**.
- [Simon Willison](https://simonwillison.net/2026/Apr/22/claude-code-confusion/) | [The Register](https://www.theregister.com/2026/04/22/anthropic_removes_claude_code_pro/) | [The New Stack](https://thenewstack.io/anthropic-claude-code-limits/)

### Google Cloud Next — Vertex AI → "Gemini Enterprise Agent Platform"으로 리브랜딩
Google이 Cloud Next 2026에서 **Vertex AI를 "Gemini Enterprise Agent Platform"으로 리브랜딩**했다. 에이전트 개발, 오케스트레이션, DevOps, 보안을 하나의 플랫폼으로 통합. Model Garden에서 **200개 이상의 모델**(Gemini 3.1 Pro, Claude Opus/Sonnet/Haiku, Gemma 4 등)에 접근 가능. **Workspace Studio**(노코드 에이전트 빌더)와 **Project Mariner**(웹 브라우징 에이전트)도 공개.

**왜 주목할만한가:** Google이 "AI 플랫폼"에서 "에이전트 플랫폼"으로 방향을 완전히 전환했다. 에이전트가 단순 챗봇이 아니라 **실제 업무를 수행하는 자율 시스템**으로 발전하고 있다는 업계 합의가 형성되고 있다. Anthropic의 Claude Managed Agents, OpenAI의 Codex와 같은 방향.
- [Google Cloud Blog](https://cloud.google.com/blog/products/ai-machine-learning/introducing-gemini-enterprise-agent-platform) | [The Next Web](https://thenextweb.com/news/google-cloud-next-ai-agents-agentic-era) | [Axios](https://www.axios.com/2026/04/22/google-unifies-gemini-enterprise-debuts-chips)

### GitHub, Copilot 가입 일시 중단 — AI 코딩 수요 폭증
**GitHub가 Copilot 신규 가입을 일시 중단**했다. AI 코딩으로 인한 컴퓨팅 수요가 공급을 초과한 것이 원인. Snap의 "AI 코드 65%"에 이어, AI 코딩 도구의 폭발적 성장을 보여주는 또 다른 지표.

**왜 주목할만한가:** AI 코딩 도구 시장이 과열 상태. Copilot 가입 중단, Claude Code Pro 제거 실험, Cursor $600억 딜 — 모두 같은 이야기를 하고 있다. **AI 코딩 도구에 대한 수요 > 공급**인 시대. 안정적인 도구 접근을 위해 유료 플랜 유지가 중요해지고 있다.

### Kyber 랜섬웨어 — "포스트양자 암호화" 주장하며 Windows + ESXi 공격
새로운 **Kyber 랜섬웨어**가 Windows 시스템과 VMware ESXi를 동시에 공격하고 있다. "Kyber1024 포스트양자 암호화"를 사용한다고 주장하지만, Rapid7 분석 결과 **Linux 변종은 실제로는 ChaCha8 + RSA-4096**을 사용하며 포스트양자 암호화는 허위 주장. Windows 변종(Rust 기반)에는 Hyper-V 공격 기능도 포함.

**왜 주목할만한가:** 포스트양자 암호화를 마케팅 도구로 쓰는 랜섬웨어의 등장. "복호화 불가능"이라는 공포를 조장해 몸값 지불을 유도하는 전략. ESXi + Windows 동시 공격은 기업 가상화 인프라 전체를 노리는 것.
- [BleepingComputer](https://www.bleepingcomputer.com/news/security/kyber-ransomware-gang-toys-with-post-quantum-encryption-on-windows/)
