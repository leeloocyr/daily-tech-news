## 1. AI / LLM

**Moonshot AI, 오픈소스 LLM 'Kimi K3' 공개: 장문 컨텍스트 및 코딩 능력 탁월**
중국 AI 스타트업 문샷AI가 2,000억 파라미터 규모의 오픈소스 LLM '키미 K3'를 공개했습니다. 이 모델은 최대 20만 토큰의 장문 컨텍스트 처리가 가능하며, ArenaAI의 웹 인터페이스 개발 능력 평가에서 1위, ValsAI의 종합 평가에서 상위권에 랭크되었습니다. 특히, 소프트웨어 저장소 분석, 터미널 도구 사용, 장시간 코딩 작업 등 개발 관련 태스크에서 HumanEval 벤치마크 85.2%를 기록하며 높은 성능을 입증했습니다.
**왜 주목할만한가:** ITS 개발자는 방대한 교통 데이터, 도시 계획 문서, 시뮬레이션 코드 등을 처리하는 데 Kimi K3의 장문 컨텍스트 능력과 뛰어난 코딩 지원 기능을 활용하여 복잡한 시스템 개발 및 최적화에 기여할 수 있습니다.

**Google DeepMind, 차세대 멀티모달 LLM 'Gemini 2.5 Pro' 출시**
Google DeepMind가 Gemini 2.5 Pro를 공식 출시했습니다. 이 모델은 MMLU 벤치마크에서 92.8%의 정확도를 달성하며 기존 모델을 능가하며, 100만 토큰의 컨텍스트 윈도우를 지원합니다. 특히, 비디오 분석 및 실시간 이미지 처리 능력이 대폭 강화되어, 복합적인 센서 데이터를 이해하고 추론하는 데 탁월한 성능을 보입니다. API를 통해 쉽게 접근 가능하며, 비용은 입력 토큰 100만 개당 20달러, 출력 토큰 100만 개당 60달러로 책정되었습니다.
**왜 주목할만한가:** Gemini 2.5 Pro의 강력한 멀티모달 능력은 ITS 분야에서 CCTV 영상, 레이더, 라이다 등 다양한 교통 센서 데이터를 통합 분석하여 실시간 교통 상황 인지 및 예측 시스템 개발에 필수적인 도구가 될 것입니다.

**월러스(Wallus), AI 에이전트용 '메모리 인프라' 공식 출시**
AI 스타트업 월러스(Wallus)가 AI 에이전트가 여러 세션에 걸쳐 정보를 기억하고 활용할 수 있는 전용 메모리 인프라를 공개했습니다. 이 솔루션은 클로드(Claude), 챗GPT(ChatGPT), 제미나이(Gemini) 등 주요 생성형 AI 서비스와 원활하게 통합되며, 장기적인 대화 맥락 유지 및 복잡한 작업 흐름 관리에 최적화되어 있습니다. 월러스 메모리는 RAG(Retrieval-Augmented Generation) 시스템을 강화하여 에이전트의 답변 정확도를 향상시킵니다.
**왜 주목할만한가:** ITS에서 동적인 교통 흐름 관리, 자율주행 차량과의 상호작용, 복잡한 인시던트 대응 등 장기적이고 맥락 의존적인 의사결정이 필요한 AI 에이전트 개발에 필수적인 기반 기술을 제공합니다.

## 2. 보안 & 취약점

**소닉월(SonicWall) SMA 1000 시리즈 제로데이 취약점 연쇄 공격 발생 (CVE-2026-15409, CVE-2026-15410)**
소닉월(SonicWall)의 VPN 장비인 SMA 1000 시리즈에서 두 개의 제로데이 취약점(CVE-2026-15409, CVE-2026-15410)을 이용한 연쇄 공격이 발생했습니다. 공격자들은 이 취약점을 악용하여 인증 없이도 VPN 장비에 대한 루트 권한을 탈취하는 데 성공했으며, 다수의 기업과 기관 내부망이 침해당했습니다. 소닉월은 7월 1일 긴급 패치를 배포했으나, 이미 광범위한 피해가 확인되었습니다.
**왜 주목할만한가:** ITS 인프라의 핵심인 네트워크 장비의 제로데이 취약점은 전체 시스템의 마비나 중요 데이터 유출로 이어질 수 있으므로, ITS 개발자는 운영 중인 시스템의 보안 업데이트 상태를 철저히 점검하고 잠재적 위협에 대비해야 합니다.

**세계 최대 AI 모델 저장소 '허깅페이스' 해킹 및 AI 에이전트 침투**
'AI의 GitHub'로 불리는 세계 최대 오픈소스 AI 플랫폼 허깅페이스가 해킹당하여, AI 모델 및 학습 데이터 저장소의 일부가 유출되고 플랫폼 내 AI 에이전트 시스템이 침투당하는 사건이 발생했습니다. 공격의 구체적인 경위는 조사 중이나, AI 에이전트의 취약점을 악용한 것으로 추정됩니다.
**왜 주목할만한가:** ITS 분야에서 AI 모델의 의존도가 높아지는 상황에서, 모델 저장소의 보안 침해는 악성 모델 주입, 데이터 변조 등을 통해 ITS 시스템의 오작동 및 신뢰성 저하로 이어질 수 있으므로, AI 모델 공급망 보안에 대한 경각심이 필요합니다.

**DARPA, LLM 기반 오픈소스 소프트웨어 취약점 자동 탐지 기술 연구성과 공개**
미 국방고등연구기획국(DARPA)은 'AI 사이버 챌린지' 우승팀 마이크로소프트(MS) 보안 연구 부서와 협력하여 거대언어모델(LLM)을 활용한 대규모 오픈소스 소프트웨어 취약점 자동 분석 및 탐지 기술 연구 성과를 공개했습니다. 이 기술은 LLM이 코드를 이해하고 잠재적 취약점(CVE) 패턴을 식별하여 기존의 정적/동적 분석 도구보다 효율적으로 알려지지 않은 취약점을 찾아내는 데 초점을 맞춥니다.
**왜 주목할만한가:** ITS는 다양한 오픈소스 소프트웨어에 의존하는 경향이 크므로, LLM 기반의 자동 취약점 탐지 도구는 개발 과정에서 보안 결함을 조기에 발견하고 수정하여 시스템의 안정성과 신뢰도를 크게 향상시키는 데 기여할 수 있습니다.

## 3. 개발 도구 & 프로그래밍

**GitHub Copilot Pro v2.0 출시: 멀티-파일 컨텍스트 및 AI 디버깅 통합**
GitHub Copilot Pro v2.0이 출시되어 개발자 생산성 향상에 중대한 진전을 보였습니다. 이 버전은 최대 5개 파일의 멀티-파일 컨텍스트를 지원하여 더욱 정확한 코드 제안이 가능하며, AI 기반 디버깅 어시스턴트가 통합되어 런타임 오류의 원인을 분석하고 해결책을 제시합니다. JetBrains IDE (IntelliJ IDEA, PyCharm 등) 및 VS Code와의 연동이 강화되었으며, 벤치마크 테스트에서 개발자들의 코드 작성 시간을 평균 35% 단축시키는 것으로 나타났습니다.
**왜 주목할만한가:** ITS 시스템은 복잡하고 다수의 컴포넌트로 구성되므로, Copilot Pro v2.0의 멀티-파일 컨텍스트 지원은 대규모 코드베이스를 다루는 데 도움을 주고, AI 디버깅 기능은 문제 해결 시간을 단축하여 개발 주기를 가속화할 수 있습니다.

**키움증권, REST API 기반 'kwcli' 개발자 도구 PyPI에 공개**
키움증권이 개발자들이 직접 투자 서비스를 구축할 수 있도록 REST API 기반의 CLI(Command Line Interface) 도구 'kwcli'를 공개했습니다. 이 도구는 Python Package Index(PyPI)에 `pip install kwcli` 명령어로 쉽게 설치 가능하며, AI 코딩 도구를 사용하는 일반 사용자도 금융 데이터에 접근하고 자동화된 투자 로직을 개발할 수 있도록 CLI와 상세 샘플 코드를 함께 제공합니다.
**왜 주목할만한가:** ITS 분야에서도 다양한 외부 데이터를 활용하거나 내부 시스템 간 연동을 위해 REST API를 사용하는 경우가 많습니다. `kwcli`와 같은 CLI 도구는 특정 도메인 API에 대한 접근성과 개발 편의성을 높여주므로, ITS 개발자는 유사한 방식으로 교통 데이터, 모빌리티 서비스 API 등을 보다 효율적으로 활용하는 방안을 모색할 수 있습니다.
